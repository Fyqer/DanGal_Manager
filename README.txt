##########README############(PL)

1. CZYM JEST DAN-GAL?

Dan-gal to menadżer haseł który nie przechowuje haseł nigdzie indzej niż na Twoim komuterze. Program nie potrzebuje połączenia z internetem. Pozwala szyfrować hasła trybem AES256 oraz główne hasło hashem MD5.

2.URUCHAMIANIE:
Program nie potrzebuje instalacji dodatkowych bibliotek. Aby uruchomić program należy użyc pliku wyknowywalnego ( .exe ).
- program poprosi nas o podanie głównego hasła, jeżeli wcześniej nie korzystaliśmy pojawi się nowy plik masterPassword z hashem głównego hasła.
- następepnie nalezy ponownie uruchmic program i zacząć z niego korzystać


3. Częste problemy
- program nie posiada edycji aktualnych haseł, aby zastąpić hasło nowym należy usunąć stare i dodać nowe z nazwą poprzedniego serwisu.
- czasami występuję błąd w szyfrowaniu hasła i powstają znaki nie  możliwe do odczytania przez człowieka. Należy wtedy wygenerować nowe hasło.
- aby zmienic główne hasło należy usunąć plik masterPassword.txt. UWAGA! - nie ma możliwości zmiany głównego hasła bez utraty wszystkich zaszfyrowanych haseł.
